1 

 < ol >

Note

  1. The HTML < ol> element is found within the < body> tag.
  2. The < ol> tag is made up of < li> tags.
  3. Use the < ol> tag when the list items have numeric, meaningful ordering.
  4. You can nest < ol>, < ul> and < menu> lists.

Browser Compatibility

The < ul> tag is compatible with the following browsers:

  1. Chrome
  2. Android
  3. Firefox (Gecko)
  4. Firefox Mobile (Gecko)
  5. Internet Explorer (IE)
  6. Edge Mobile
  7. Opera
  8. Opera Mobile
  9. Safari (WebKit)
  10. Safari Mobile
2 

 < ul >

Note

  • The HTML < ul > element is found within the < body> tag.
  • The < ul> tag is made up of < li> tags.
  • Use the < ul> tag when the list items have no numerical ordering.
  • You can nest < ol>, < ul> and < menu> lists.

Browser Compatibility
